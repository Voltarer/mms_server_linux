# MMS Server 

Минимальный проект MMS-сервера (IEC 61850) с интеграцией аппаратных
портов коммутатора Realtek.

![Version](https://img.shields.io/badge/version-2.1.1-blue)

## История изменений

<details>

## Структура проекта
* **`src/`** — Исходный код сервера.
    * `main.c` — Основная логика: опрос портов и обновление модели.
    * `model.icd` — Описание логической модели устройства (SCL).
    * `static_model.c/h` — Файлы, генерируемые из ICD для работы сервера.
* **`lib/`** — Библиотеки и SDK.
    * `libiec61850/` — Стек протоколов МЭК 61850.
    * `sdk/` — Заголовочные файлы и библиотеки Realtek.
* **`build/`** — Папка с готовым бинарным файлом.

## 🛠 Скрипты управления

| Скрипт | Назначение |
| :--- | :--- |
| `./gen.sh` | **Генератор модели.** Превращает XML (`.icd`) в Си-код. Запускать при изменении состава портов или сигналов. |
| `./build.sh` | **Компилятор.** Собирает бинарник под архитектуру MIPS. |
| `./setup.sh` | **Подготовка к сборке.** Скачивает нужные для работы сервера файлы, собирает библиотеку под MIPS


## Быстрый старт

``` bash
chmod +x setup.sh
./setup.sh
```

### 3. Сборка под MIPS

``` bash
chmod +x build.sh
./build.sh
```
